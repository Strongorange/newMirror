# Smart Mirror

날씨, 시간, 미세먼지, Firestore에 업데이트한 사진 정보를 보여주는 스마트 미러.
하드웨어와 한 세트

## 기술 스택

- React
- Firebase
- [OpenWeatherMap](https://openweathermap.org/)
- [한국환경공단 에어코리아 대기오염정보](https://www.data.go.kr/tcs/dss/selectApiDataDetailView.do?publicDataPk=15073861)
